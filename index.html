1.¿Qué es CSS?

CSS son las siglas de “Cascading Style Sheets” (hojas de estilo en cascada). CSS es un lenguaje para la composición y estructuración de páginas web (HTML o XML). Este lenguaje contiene elementos de codificación y se compone de estas “Cascading Style Sheets” que también se llaman archivos CSS (. css).


2.¿Cuál es la función principal de CSS?

Como hemos mencionado, el CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc. Un documento suele ser un archivo de texto estructurado con un lenguaje de marcado: HTML es el más común, pero también existen otros como SVG o XML.


3.¿Cómo se puede incluir CSS en un documento HTML?

Cuando hablamos de estilo en línea, pretendemos insertar CSS en una sola línea: la misma que el elemento que vamos a modificar. En este caso crearemos un atributo global, el atributo "style" dentro de la etiqueta html, seguido de la propiedad css que vamos a modificar y el valor de esa propiedad.


4.¿Qué son los selectores en CSS?

Un selector CSS es la primera parte de una regla CSS. Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS.


5.¿Qué son las propiedades y valores en CSS?

Las propiedades CSS no son más que aquellas características a las que vamos a asignar valores que definirán el estilo concreto de nuestros elementos html.


6.¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja en CSS describe cómo se estructuran los elementos en una página web. Cada elemento está compuesto de cuatro áreas: el contenido, el padding (relleno), el borde y el margin (margen). Estos determinan el tamaño total y el espacio que ocupa un elemento en la página.


7.¿Qué es el "cascading" en CSS?

El "cascading" en CSS se refiere a cómo se resuelven los conflictos cuando múltiples reglas se aplican a un mismo elemento. Las reglas se priorizan según su origen (hoja externa, interna o inline), la especificidad del selector y el orden en que aparecen. Las más específicas o que están más abajo en el código suelen prevalecer.


8.¿Qué son las media queries?

Las media queries en CSS son reglas que permiten aplicar estilos específicos según las características del dispositivo, como el tamaño de la pantalla, la resolución o la orientación. Son clave para crear diseños responsivos que se adapten a diferentes tipos de pantallas (móviles, tabletas, desktops). Por ejemplo, puedes usar una media query para cambiar el diseño si el ancho de la pantalla es menor a 768px.


9.¿Cuáles son algunos estilos comunes que se pueden aplicar?

Color y fondo: color: cambia el color del texto. background-color: establece el color de fondo. background-image: aplica una imagen de fondo. Tipografía: font-size: ajusta el tamaño del texto. font-family: define la fuente. font-weight: establece el grosor de la letra (negrita). Márgenes y relleno: margin: define el espacio exterior alrededor de un elemento. padding: controla el espacio interior entre el contenido y el borde. Bordes y sombras: border: define el estilo, ancho y color del borde. box-shadow: agrega sombras a los elementos. Posicionamiento y diseño: display: controla cómo se muestra un elemento (block, inline, flex). position: define la posición (relative, absolute, fixed). width y height: ajustan el ancho y alto de un elemento. Flexbox y Grid: display: flex y display: grid: crean diseños más complejos y responsivos.


10.¿Cómo se puede usar CSS para crear un diseño responsivo?

Para crear un diseño responsivo con CSS, se utilizan media queries para aplicar estilos adaptados a diferentes tamaños de pantalla. También se emplean unidades flexibles como %, em o rem en lugar de píxeles fijos, y técnicas como flexbox y CSS grid para ajustar la disposición de los elementos en diferentes dispositivos.
